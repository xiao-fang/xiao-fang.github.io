WordPress on Azure - S3 - Setup WordPress on Azure VM

Posted on 2016-04-24 | In 学以致用 , WordPress | | Visitors

WordPresss Azure WordPress Ubuntu PuTTY posts

1. Create MySQL database via PhpMyAdmin
Goto PhpMyAdmin site (http://vm-host/phpmyadmin/), login with root user and password.

Create a new database, give an unique database name, choose utf8-general-ci, then Create.

2. Set Up WordPress
Goto WordPress site http://vm-host/wp-admin/setup-config.php for a first-time installation.

Configure database connection details for WordPress, with MySQL database just created.

Database connected, then go ahead to run the install.

Quick configuration for WordPress site.

Success, enjoy!
