WordPress Azure WordPress Ubuntu PuTTY

Azure is an open, flexible, enterprise-grade cloud computing platform, provided by Microsoft, one of the best and popular public cloud platform.

1. Login to Azure Portal
Goto Azure Portal (https://portal.azure.com/) and login with your account, then you can see a dashboard.

2. Configure and Create New Ubuntu VM
Click + New button on the left sidebar, then click Virtual Machines category, scroll down next open Virtual Machines list and select Ubuntu Server 14.04 LTS, also we can Search the market with keyword ‘Ubuntu’ to find this VM.

Then we can choose Classic mode, click Create.

learn more about deployment model, the difference between Classic and Resource Manager, refer to http://go.microsoft.com/fwlink/?LinkId=618337

Quick configuration before creating VM, here we should input Host Name, User name, and basic authentication type Password. Choose a proper pricing tier, here I choose a very cheap Standard A1 tier. And Avaliability set is optional here I left it as default, Network also default settings we can see Virtual Network, Subnet, even Domain Name for Network, then choose a Location, click Create, it may take about 2 minutes to create and deploy new VM on Azure.

Okay, as we can see a new Ubuntu VM is now created successfully!

We can get detail VM settings, such as DNS Name, Virtual IP, also an Endpoint named SSH (Port 22) was set enabled as default, with SSH we can remote connect to VM.

3. Remote to Ubuntu VM with PuTTY
We can remote connect to Ubuntu server with client PuTTY.
A first time connection with PuTTY, a certificate warning will prompt, click YES to accept then login with user name and password.

So here we can now create a new Ubuntu VM on Auzre, and remote login with 3rd-party client like PuTTY successfully. Well Done.
