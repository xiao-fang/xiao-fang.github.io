### `git log`

here is my `git log pretty scheme`.
```bash
git config --global alias.hist "log --pretty=format:'%C(yellow)[%ad]%C(reset) %C(green)[%h]%C(reset) | %s %C(bold red)%d%C(reset) %C(green)[%an]%C(reset)' --graph --date=short"
```
